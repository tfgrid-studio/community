## Quick Start

### New to TFGrid Studio?

1. **Visit the website:** [tfgrid.studio](https://tfgrid.studio)
2. **Read the docs:** [docs.tfgrid.studio](https://docs.tfgrid.studio)
3. **Get the CLI:** [Install TFGrid Compose](https://docs.tfgrid.studio/getting-started/installation)

### Deploy Your First App

```bash
# Install TFGrid Compose
git clone https://github.com/tfgrid-studio/tfgrid-compose
cd tfgrid-compose
make install

# Deploy the AI Agent (2 minutes!)
tfgrid-compose up tfgrid-ai-agent

# You're live! 
```
